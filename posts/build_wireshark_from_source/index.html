下载源码

下载源码

git clone https://gitlab.com/wireshark/wireshark.git

安装依赖项

sudo apt-get install flex g++ gcc perl python3 cmake build-essential -y

执行设置脚本

tools/debian-setup.sh --install-optional --install-deb-deps

安装

cd wireshark
cmake .
make
make install

启动

wireshark
