批量删除docker镜像

###删除mariadb的所有镜像
###方法1
docker images|grep mariadb | awk 'BEGIN{OFS=":"} {print $1,$2}' | while read line; do
docker rmi -f $line
done
###方法2：不用read line也可以做到的
docker rmi -f `docker images |grep mariadb |awk '{print $3}'`
